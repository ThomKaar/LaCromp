# LaCromp
LaCromp is a website for tracking a group's LaCroix consumption and analyzing consumption trends.

***
## Key Stories

* Create A Group
* Login/Create Account
* Groups can compare users LaCroix consumption
* Users can enter LaCroix they have drank
